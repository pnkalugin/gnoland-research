# gno.land — Глубокое исследование проекта & $GNOT

Интерактивный аналитический отчёт по проекту gno.land: технология, токеномика $GNOT, аirdrop-чекер, сравнение с конкурентами, прогноз.

## Разделы

1. **Обзор** — что такое gno.land, три проблемы которые решает, ключевые фичи
2. **Технология** — GnoVM, Gnolang vs Solidity/Rust, архитектура
3. **Airdrop** — чекер GNOT, условия eligibility, как проверить баланс
4. **Токен** — утилити $GNOT, токеномика, сценарии оценки цены
5. **Конкуренты** — сравнение с Ethereum, Solana, Cosmos, Aptos, Near
6. **Комьюнити** — настроения в X/Twitter, Discord, критика и поддержка
7. **Прогноз** — роадмап, риски, финальный вердикт
